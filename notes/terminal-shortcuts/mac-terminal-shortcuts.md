# Mac 终端常用快捷键操作指南

## 光标移动

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + A` | 跳到行首 |
| `Ctrl + E` | 跳到行尾 |
| `Ctrl + F` | 向右移动一个字符 |
| `Ctrl + B` | 向左移动一个字符 |
| `Option + ←` | 向左跳一个词 |
| `Option + →` | 向右跳一个词 |

## 删除与清空

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + U` | 清空光标前的所有内容（最常用，一步清空整行） |
| `Ctrl + K` | 清空光标后的所有内容 |
| `Ctrl + W` | 删除光标前的一个单词 |
| `Ctrl + H` | 删除光标前的一个字符（等同 Backspace） |
| `Ctrl + D` | 删除光标后的一个字符（等同 Delete） |

> 清空整行最快的方式：`Ctrl + A` → `Ctrl + K`，或直接 `Ctrl + U`

## 文本选择

| 操作 | 功能 |
|------|------|
| 鼠标三击 | 选中整行 |
| `Cmd + A` | 全选终端所有可见内容 |

## 复制与粘贴

| 快捷键 | 功能 |
|--------|------|
| `Cmd + C` | 复制选中文本 |
| `Cmd + V` | 粘贴文本 |
| `Ctrl + Y` | 粘贴被 Ctrl+U/K/W 剪切的内容 |

## 历史命令

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + R` | 搜索历史命令 |
| `↑` / `↓` | 浏览上一条 / 下一条历史命令 |
| `Ctrl + P` | 同 ↑，上一条历史命令 |
| `Ctrl + N` | 同 ↓，下一条历史命令 |
| `!!` | 执行上一条命令 |
| `!$` | 引用上一条命令的最后一个参数 |

## 进程控制

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + C` | 终止当前运行的程序 |
| `Ctrl + Z` | 挂起当前程序（用 `fg` 恢复） |
| `Ctrl + D` | 退出当前 Shell（等同 `exit`） |
| `Ctrl + L` | 清屏（等同 `clear`） |

## 屏幕操作

| 快捷键 | 功能 |
|--------|------|
| `Cmd + K` | 清空终端滚动缓冲区 |
| `Cmd + Shift + K` | 清空当前终端窗口 |
| `Ctrl + L` | 清屏（保留滚动历史） |

## 标签页与窗口

| 快捷键 | 功能 |
|--------|------|
| `Cmd + T` | 新建标签页 |
| `Cmd + W` | 关闭当前标签页 |
| `Cmd + N` | 新建终端窗口 |
| `Cmd + 数字` | 切换到对应标签页 |
| `Cmd + Shift + [` / `]` | 切换上一个 / 下一个标签页 |
